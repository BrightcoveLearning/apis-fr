Champs de profil

Biens Taper La description Défaut
account_id entier Les Nuage vidéo identifiant de compte rien
name chaîne de caractères Le nom du profil - doit être unique dans le compte rien
display_name chaîne de caractères Le nom d'affichage du profil à afficher sur la page d'administration de Studio - peut être le même que name , mais doit être descriptif pour aider les utilisateurs à comprendre l'utilisation prévue du profil rien
description chaîne de caractères La description du profil rien
model_version entier Réglez ceci sur 1 pour faire apparaître le profil dans le Live Module rien
renditions déployer Tableau d'objets de rendu []
packages déployer Tableau de packages DRM - actuellement, DRM n'est pas pris en charge, il doit donc toujours s'agir d'un tableau vide []

Champs de rendu

Les tableaux ci-dessous présentent les champs qui peuvent être utilisés dans le corps de la demande pour mettre à jour les profils d'ingestion en direct.

Biens Taper La description Défaut
audio_bitrate entier débit audio en kbps source de correspondance
audio_codec chaîne de caractères Un codec audio - aac | mp3 Déterminé par le format ou le profil< - dans la plupart des cas, aac sera le meilleur/td>
decoder_bitrate_cap entier en kbps, le débit binaire maximal fourni au décodeur via un tampon rien
decoder_buffer_size entier en kbps, la taille de la mémoire tampon fournie au décodeur lors de l'utilisation d'un bitrate_cap rien
forced_keyframe_rate numéro Force une image clé toutes les X images, mais autorise toujours des images clés supplémentaires pour la coupe de scène, etc. rien
format chaîne de caractères Format vidéo - ts , mp4 , flv rien
h264_profile chaîne de caractères H.264 a trois profils couramment utilisés : Baseline (le plus bas), Main , et High.[1-5] baseline
hls_optimized_ts booléen Segments temporels optimisés pour HLS - s'applique uniquement à la sortie HLS rien
keyframe_interval entier Nombre maximum d'images entre les images clés 250
keyframe_rate flotter Nombre maximal d'images clés par seconde keyframe_interval de 250
label chaîne de caractères Type d'image pour les rendus d'image - poster ou thumbnail rien
live_stream booléen si ce sera un flux vidéo en direct rien
live_dvr_sliding_window_duration entier Durée du flux à garder disponible pour la livraison LiveDVR en quelques secondes 300
max_video_bitrate entier en kbps, 123 = 123kbps - h264 uniquement rien
max_frame_rate flotter limite le frame_rate plutôt que de le définir. Doit être défini sur 30 sauf si votre compte a été activé par le support pour 60 FPS rien
media_type chaîne de caractères video ou audio rien
video_bitrate entier en kbps, 123 = 123kbps rien
video_codec chaîne de caractères par exemple h264, ts Déterminé par le format ou le profil
width entier largeur du cadre en pixels rien
height entier hauteur du cadre en pixels rien
