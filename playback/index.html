--- title: API de lecture parent: Home description: '

L'API de lecture est utilisée pour récupérer les données de vidéo et de liste de lecture à partir de Video Cloud. Il est généralement utilisé pour transmettre des données vidéo à un lecteur sur une page Web ou dans une application mobile. L'API de lecture est une API à faible latence destinée à être utilisée côté client pour récupérer des données de vidéo ou de liste de lecture à partir de pages Web ou d'applications mobiles. Il ne s'agit pas d'une API générale de gestion des médias à utiliser pour intégrer Video Cloud à votre CMS ou à d'autres systèmes - pour cela, vous devez utiliser le API CMS.

L'authentification pour l'API Playback requiert une clé de stratégie, que vous pouvez obtenir à l'aide du API de règles ou d'un joueur sur ton compte.
