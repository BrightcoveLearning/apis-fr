--- title: Playback API parent: Home description: "

le Playback API est utilisée pour récupérer les données de vidéo et de liste de lecture à partir de Video Cloud. Il est généralement utilisé pour transmettre des données vidéo à un lecteur sur une page Web ou dans une application mobile. le Playback API est une API à faible latence destinée à être utilisée côté client pour récupérer des données de vidéo ou de liste de lecture à partir de pages Web ou d'applications mobiles. Il ne s'agit pas d'une API générale de gestion des médias à utiliser pour intégrer Video Cloud à votre CMS ou à d'autres systèmes - pour cela, vous devez utiliser le CMS API.

L'authentification pour le Playback API requiert une clé de stratégie, que vous pouvez obtenir à l'aide du Policy API ou d'un Lecteur sur ton compte.
