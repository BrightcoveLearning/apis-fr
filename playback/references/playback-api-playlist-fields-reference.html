Champs principaux

Tableau 1: Champs principaux
Champ Taper Défaut La description
id chaîne de caractères rien l'identifiant de la playlist
account_id chaîne de caractères rien l'identifiant du compte
created_at Chaîne date-heure ISO 8601 rien la vidéo date-heure a été ajoutée au compte ; Exemple: "2014-12-09T06:07:11.877Z"
description chaîne de caractères null la brève description de la vidéo - 128 caractères mono-octet maximum
favorite booléen false si la liste de lecture est incluse dans les favoris
limit numéro 100 nombre de vidéos que la playlist peut contenir (entier, 1 - 100) renvoyé pour les listes de lecture intelligentes uniquement
name chaîne de caractères rien Titre de la liste de lecture - champ obligatoire
reference_id chaîne de caractères null toute valeur unique dans le compte
search chaîne de caractères null chaîne de recherche utilisée pour définir les vidéos d'une liste de lecture Smart (exemple: +tags:"bird","birds")
type chaîne de caractères le type de liste de lecture Voir la note[1] au dessous de
updated_at Chaîne date-heure ISO 8601 rien la liste de lecture date-heure a été modifiée pour la dernière fois ; Exemple: "2015-01-13T17:45:21.977Z"
videos déployer tableau de objets vidéo pour les vidéos de la playlist seules les 20 premières vidéos sont renvoyées pour les playlists

Remarques

  • [1]

    Les type la valeur ne peut être que l'une des suivantes: EXPLICITE, ACTIVATED_OLDEST_TO_NEWEST , ACTIVATED_NEWEST_TO_OLDEST , ALPHABETICAL , PLAYS_TOTAL , PLAYS_TRAILING_WEEK , START_DATE_OLDEST_TO_NEWEST,START_DATE_NEWEST_TO_OLDEST

    EXPLICIT signifie qu'il s'agit d'une liste de lecture manuelle ; tous les autres types sont des listes de lecture intelligentes avec des noms correspondant à leur schéma de commande.
