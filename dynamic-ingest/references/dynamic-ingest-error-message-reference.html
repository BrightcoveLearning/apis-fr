Messages d'erreur

Les messages d'erreur suivants peuvent être renvoyés par l'API Dynamic Ingest

Messages d'erreur d'ingestion dynamique
Code Un message Taper La description
400 PROFILE json

Impossible de trouver le profil par nom
400 BAD_REQUEST json

Impossible d'analyser le corps de la demande.
400 NOT_SUBMITTED json

Impossible de soumettre le travail, veuillez réessayer plus tard
400 NO_SUCH_VIDEO json

Impossible de trouver la vidéo référencée
400 NO_SOURCE json

Impossible de trouver une source à utiliser
400 CDN_CREDENTIALS json

Impossible de récupérer les identifiants CDN
400 BAD_CALLBACKS json

Les rappels n'étaient pas dans le format attendu
401 UNAUTHORIZED json

Authentification échouée; vérifiez que vos informations d'identification client étaient correctes pour le jeton d'accès
403 DYNAMIC_DELIVERY_NOT_ALLOWED json

Ce compte n'est pas activé pour la livraison dynamique, mais un profil de livraison dynamique a été spécifié
403 CONTEXT_AWARE_ENCODING_NOT_ALLOWED json

Ce compte n'est pas activé pour le codage contextuel, mais un profil de codage contextuel a été spécifié
404 RESOURCE_NOT_FOUND json

L'API n'a pas pu trouver la ressource que vous avez demandée
415 UNSUPPORTED_MEDIA_TYPE json

Soit le fichier que vous essayez d'ingérer n'est pas d'un type pris en charge (voir Formats, codecs et conteneurs vidéo et audio pris en charge ) ou la demande ne comportait pas de Content-Type: application/json entête
422 MALFORMED_SOURCE_URL json

l'URL source est mal formée.
422 BAD_PROTOCOL_SOURCE_URL json

L'URL source utilise un protocole non pris en charge.
422 EXCEED_MAXIMUM_VTT_SOURCES json

les sources vtt dépassent la taille maximale.
422 INVALID_VTT_KIND json

Le type vtt n'est pas valide.
422 CONSTRAINT_VIOLATION json

capture-image n'est pas autorisé si une source d'image est fournie.
422 UNPROCESSABLE_ENTITY json

les données de la demande contiennent une entité non traitable.
429 RATE_LIMIT_EXCEEDED json

La tâche d'ingestion dynamique n'a pas été créée. Réduisez le nombre de tâches simultanées pour ce compte avant de réessayer. Cette erreur a été enregistrée
400 CDN_CONFIGS json

Impossible de récupérer les identifiants CDN
400 AMBIGUOUS_REQUEST json

Une URL principale et use_archived_master ont été définies dans la demande.
500 INGEST_ERROR json

Cette erreur indique généralement qu'il y a eu une interruption temporaire du service d'ingestion.
