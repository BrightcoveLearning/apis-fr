introduction

Le planificateur en direct vous permet de planifier des actions pour Tâches de point d'entrée statique (SEP).. Vous venez de configurer un workflow de planificateur avec les heures, et les actions API nécessaires seront gérées automatiquement. Il existe deux types de workflows de planificateur que vous pouvez configurer, comme détaillé dans les sections suivantes.

Planificateur de démarrage/arrêt

Le planificateur vous permet de configurer des heures d'activation et de désactivation pour un travail SEP, et vous pouvez également mettre à jour les heures. Voir les détails dans Planification de l'activation/désactivation d'un flux SEP.

Planificateur de clips

Vous pouvez également planifier la génération d'un clip à partir de votre travail en direct, puis mettre à jour la planification ultérieurement ou annuler le clip si quelque chose change. Voir les détails dans Planification de la génération de clips pour une diffusion en direct.
