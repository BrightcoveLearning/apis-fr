Ingestion

Pour ajouter un fichier audio à votre compte Video Cloud, utilisez le API d'ingestion dynamique ou la Module de téléchargement. Votre fichier source peut être au format MP3, M4A ou WAV. Le processus d'ingestion nécessite deux requêtes :

  1. Une demande de création au API CMS pour créer l'article dans votre catalogue Video Cloud
  2. Une requête d'ingestion à l'API Ingest pour ingérer et transcoder le fichier

Les détails de ces demandes se trouvent dans le Référence de l'API d'ingestion dynamique.

Profil d'ingestion

Livraison dynamique

Si votre compte est activé pour la diffusion dynamique, vous n'avez pas besoin d'un profil personnalisé pour le contenu uniquement audio.

Acquisition héritée

Si votre compte utilise le système d'acquisition hérité, utilisez un profil qui crée des rendus audio uniquement. Vous pouvez utiliser le audio-only profil qui est inclus dans le Profils standards , ou vous pouvez créer un profil personnalisé dans le Paramètres du compte Studio ou en utilisant le Ingérer l'API des profils.

Rendus

Les rendus créés dans Video Cloud pour votre élément audio seront au format MP4 et HLS, avec une piste audio uniquement. L'élément peut être lu dans un Brightcove Player comme une vidéo, comme expliqué au dessous de.

Noter: Afin de maximiser la couverture de la lecture sur plusieurs plates-formes/navigateurs, nous recommandons que les fichiers audio soient transcodés en rendus MP4 et HLS avec des pistes audio uniquement - vous ne pouvez pas spécifier MP3 comme format de sortie.

Affiche et vignette

En option, vous pouvez ajouter une affiche et/ou une image miniature à votre fichier audio. L'affiche s'affichera dans le lecteur Brightcove pendant la lecture et la vignette s'affichera dans toute liste de lecture à laquelle vous ajoutez l'élément.

Pour ajouter des images à votre élément audio, vous pouvez soit modifier les paramètres des éléments dans Studio ou utilisez le API d'ingestion dynamique.

Lecteur audio

Vos éléments audio seront lus dans n'importe quel Lecteur Brightcove. Pendant la lecture, l'écran vidéo sera soit vide, soit affichera l'image de l'affiche si vous ajouté un.

Si vous préférez utiliser un lecteur audio plus minimal sans l'écran vidéo, vous pouvez créer un lecteur à cet effet. Après Vous créer le joueur , configurer la taille De la manière suivante:

  • Utiliser une taille fixe
  • Utiliser un rapport hauteur/largeur personnalisé
  • Réglez la hauteur sur 80px (la largeur peut être ce que vous voulez)
