introduction

Forensic Watermarking (FW) vous permet de protéger votre contenu premium contre le piratage et le partage de contenu non autorisé. Cette fonctionnalité ajoute un filigrane invisible à vos vidéos, vous permettant de suivre toute fuite de contenu.

Pour en savoir plus sur cette fonctionnalité, consultez le Aperçu: Filigrane médico-légal document.

Général

Comment fonctionne la détection de contenu ?

Le processus de détection est basé sur une partie du contenu qui peut être téléchargée, capturée ou enregistrée dans un fichier. Un outil spécifique de NAGRA, appelé Conditionneur , génère un .pak déposer. Ce fichier est soumis à NAGRA pour récupérer la séquence unique livrée, qui est ensuite associée à un utilisateur. Vous êtes responsable de faire correspondre la séquence à l'utilisateur. Les services de détection sont acquis directement auprès de NAGRA ; généralement effectué par un tiers engagé par le propriétaire du contenu pour explorer Internet afin de trouver du contenu illégal.


Intégrations

Le filigrane judiciaire est-il pris en charge avec des appareils tels que Chromecast ou AirPlay ?

Oui, le jeton de filigrane est transmis au récepteur de diffusion. Ceci est pris en charge avec le lecteur Web et les SDK natifs.


Suivi et rapports

Comment NAGRA suit-il le nombre de transactions utilisées par un client ?

NAGRA n'a aucun moyen de savoir si une partie de notre logiciel est connectée à NAGRA pour déclarer les transactions.

Vous êtes responsable de télécharger depuis l'API Analytics un rapport avec le nombre de visiteurs uniques par jour, toutes les 24h, pour fournir comme preuve le nombre de transactions utilisées.


Comment les rapports sont-ils gérés ?

Les rapports d'analyse sur les téléspectateurs pour les vidéos avec filigrane médico-légale sont disponibles via l'API Analytics.

Pour plus de détails, consultez le Aperçu: Filigrane médico-légal document.
