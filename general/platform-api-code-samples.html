    Dans cet exemple, vous entrez les informations de votre compte. De plus, vous entrez une URL vers le média et le type de média, ou un ID vidéo Video Cloud. En cliquant sur le bouton, un joueur sera créé et sélectionné des informations sur le joueur affiché.

    Statistiques des derniers joueurs consultés

    Rapports utilisant les last_viewed données du joueur Chaque joueur lui a associé un tableau qui contient un enregistrement du nombre de fois où un joueur a été chargé. Ces informations sont stockées dans un nom de tableau last_viewed.
