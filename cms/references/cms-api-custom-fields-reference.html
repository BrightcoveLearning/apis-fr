Les champs personnalisés

Champ Taper La description
max_custom_fields Nombre Nombre de champs personnalisés qu'un éditeur peut créer sur le compte
custom_fields Déployer Liste des objets de champ personnalisés sur le compte
standard_fields Déployer Liste des champs obligatoires ou comportant une description. Les champs standard qui n'ont ni l'un ni l'autre n'apparaîtront pas dans ce tableau.

Carte de champ personnalisée

Pour chaque champ personnalisé:

Champ Taper La description
id Chaîne de caractères Nom interne du champ
description Chaîne de caractères  
display_name Chaîne de caractères  
type Chaîne de caractères l'un des 'enum' ou 'chaîne' (qui se traduit par 'Liste' ou 'Texte' dans l'interface utilisateur)
enum_values Déployer Tableau de valeurs de chaîne valides pour le champ personnalisé ; présent uniquement lorsque type = 'enum'
required booléen Si ce champ doit avoir une valeur avant que la vidéo puisse être activée
