Remarques

  • [1] Les règles suivantes s'appliquent aux noms de dossier:
    • les noms de dossier sont limités à 100 caractères
    • les noms de dossier doivent être uniques dans le compte
    • caractères illégaux: /
    • le nom ne peut pas être vide (null , vide, tous les espaces)
