    Définir les propriétés vidéo à 360 degrés

    Dans cette rubrique, vous apprendrez comment définir des propriétés vidéo à 360° à l'aide de l'API CMS. Nous utiliserons l'interface Postman pour envoyer nos demandes et consulter les réponses de l'API.

    Présentation

    Pour identifier une vidéo en tant que vidéo à 360°, la propriété de projection doit être définie. Vous pouvez définir cette propriété à l'aide du module Media dans Studio, ou vous pouvez le faire par programme comme indiqué dans cette rubrique.

    Cet exemple utilise l'application Postman pour mettre à jour les propriétés vidéo avec l'API CMS. Découvrez comment démarrer avec le document Utiliser Postman pour les requêtes HTTP .

    Dans cette rubrique, vous allez effectuer les tâches suivantes :

    • Obtenir les identifiants client
    • Récupérer des informations vidéo
    • Définir les propriétés vidéo à 360°

    Obtenir les identifiants client

    Pour travailler avec les API Brightcove, vous aurez besoin des informations d'identification client pour le compte et les API que vous souhaitez utiliser.

    1. Obtenez vos informations d'identification client dans Studio en suivant les instructions de la page Gestion des informations d'identification d'authentification API. Dans les étapes ci-dessous, nous effectuerons des demandes d'API Player Management à l'aide de Postman, de sorte que vos informations d'identification doivent avoir au moins les autorisations suivantes :

      • CMS: Lecture/écriture vidéo

      Vous pouvez ajouter autant d'autorisations supplémentaires que vous le souhaitez pour obtenir des informations d'identification qui seront utilisables pour un plus grand nombre de requêtes API. Notez également que vous obtenez des informations d'identification qui fonctionneront pour plusieurs comptes si vous le souhaitez.

      Pour générer des informations d'identification client, vous devez disposer des autorisations Admin pour Studio ou obtenir une personne de votre organisation disposant des autorisations d'administrateur pour générer les informations d'identification pour vous.

    Récupérer des informations vidéo

    Une fois que vous avez téléchargé votre vidéo sur Video Cloud Studio, vous êtes prêt à récupérer des informations vidéo à partir de l'API CMS. Pour plus d'informations sur le téléchargement de vidéos, consultez le document Téléchargement de vidéos à l'aide du module de téléchargement .

    1. Ouvrez l'application Postman.
    2. Obtenez un nouveau jeton d'accès pour votre compte Video Cloud et cliquez sur le bouton Utiliser le jeton .
    3. Effectuez une requête GET à l'API CMS, en utilisant la syntaxe Get Video by ID . Pour ce faire, saisissez l'URL suivante dans le champ Entrer l'URL de la demande , en remplaçant votre identifiant de compte par {account_id} et votre identifiant vidéo par {video_id}: 
          https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_id}
    4. Laissez la méthode comme GET et sélectionnez Envoyer:
    5. Dans le corps de la réponse, vous devriez voir la propriété de projection .

    Définir les propriétés vidéo à 360°

    Pour indiquer qu'une vidéo est une vidéo à 360°, la propriété de projection doit être définie sur équirectangulaire.

    1. Retournez à l'application Postman. Vous devrez peut-être obtenir un nouveau jeton d'accès.
    2. Sélectionnez PATCH comme méthode HTTP, en utilisant la même syntaxe URL que dans la section précédente.
    3. Cliquez sur le bouton En-têtes , et fournissez la clé - valeur paire Content-Type - application/json. Commencez simplement à taper dans les zones de saisie et vous serez fourni avec des options à partir desquelles vous pouvez choisir les valeurs indiquées.
    4. Cliquez sur la section Corps juste sous l'URL, puis sur le bouton radio brut :
    5. Pour les données brutes, entrez le code JSON suivant pour le corps (la capture d'écran suivant le JSON montre comment la requête doit apparaître) : 
          {
        "projection" : "equirectangular"
    }
    6. Cliquez sur Envoyer
    7. Dans le corps de la réponse, vous devriez maintenant voir que la valeur de la propriété de projection a été définie.

      Vous êtes maintenant prêt à lire votre vidéo en tant que vidéo à 360°.
