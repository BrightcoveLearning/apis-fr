Zencoder et Video Cloud ensemble

Vous pouvez profiter des deux plates-formes, que vous soyez un client Zencoder cherchant à utiliser Video Cloud pour déployer vos vidéos en ligne, ou un client Video Cloud souhaitant profiter de la puissance de transcodage de Zencoder.

Pour que cette solution fonctionne pour Video Cloud, vous devez disposer d'un Entreprise Compte. Les éditeurs seront également besoin de la possibilité de diffuser un actif distant depuis leur compte Video Cloud. Typiquement, cela signifie que vous aurez besoin d'une sorte de relation avec un CDN (par exemple Akamai, Limelight) où ils peuvent stocker le contenu puisqu'il ne sera pas dans le compte Brightcove House.

Les propriétaires de contenu utilisent l'API Zencoder pour télécharger du contenu sur Zencoder et créer des rendus pour mobile, Web, TV connectée, etc. Ce contenu peut ensuite être poussé dans un CDN pour être diffusé vers le lecteur Video Cloud. Dans Video Cloud, une ressource distante doit être créée via les API, le module multimédia ou les fichiers manifestes FTP. Cela vous permettra d'utiliser l'API Video Cloud Media pour obtenir les métadonnées et un pointeur vers la ressource distante sur votre CDN.

Et voici les étapes de haut niveau pour l'intégration:

Transcoder la vidéo à l'aide de Zencoder

  1. Intégrez Zencoder à votre application (CMS, web app, etc.) - voir le API et bibliothèques d'intégration bien documentées pour les détails.
  2. L'emplacement de sortie peut être n'importe quel emplacement pouvant diffuser du contenu vers le lecteur vidéo de l'éditeur. Il s'agit généralement d'un CDN comme Akamai ou Limelight.

Ajouter la vidéo à Video Cloud

  1. Créer une nouvelle vidéo dans le nuage vidéo.
  2. Si vous devez ajouter des métadonnées qui ne peuvent pas être incluses dans la demande de création de vidéo, Mettre à jour les métadonnées de la vidéo
  3. Ajoutez votre vidéo en tant que rendu à distance à la vidéo, en utilisant l'URL de votre CDN ou d'un autre référentiel. Dans la plupart des cas, vous utiliserez le Ajouter un rendu opération pour ajouter un rendu H.264 ou le Ajouter un manifeste HSL pour ajouter un rendu HLS.
