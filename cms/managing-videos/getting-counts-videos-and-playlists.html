introduction

Il est parfois utile d'obtenir le nombre de vidéos ou de listes de lecture dans votre compte, ou le nombre de vidéos ajoutées au cours du mois ou de l'année écoulée. Vous pouvez facilement obtenir ces types de comptes en utilisant le /counts/ point de terminaison pour l'API CMS.

Authentification

Demandes au API CMS nécessitent un en-tête d'autorisation contenant vos jetons d'accès. Pour plus de détails sur la façon d'obtenir les informations d'identification client et de les utiliser pour récupérer des jetons d'accès, consultez le Présentation de Brightcove OAuth.

Obtenez le nombre de vidéos

Pour obtenir le nombre de vidéos, faites un GET demande à:

    https://cms.api.brightcove.com/v1/accounts/{account_id}/counts/videos

La réponse ressemblera à ceci :

    {
      "count": 4999
    }
Notez que les vidéos supprimées ne sont pas incluses dans les décomptes, ni dans aucun API CMS réponse.

Ce qui rend le point de terminaison de comptage vidéo si puissant, c'est que vous ajoutez également un paramètre de recherche à la demande. Cela vous permet d'obtenir un décompte des vidéos que la recherche renverrait. Parce que le paramètre de recherche est si flexible, cela vous permet d'obtenir un décompte de nombreuses choses, telles que :

  • le nombre de vidéos qui ont une balise particulière ou une valeur spécifique pour un champ personnalisé
  • le nombre de vidéos ajoutées au cours de la dernière année (ou mois, ou semaine, etc.)
  • le nombre de vidéos actuellement lisibles
  • le nombre de vidéos dont la sortie est prévue la semaine prochaine

Obtenir le nombre de listes de lecture

Vous pouvez également obtenir un décompte des listes de lecture dans votre compte en faisant un GET demande à:

    https://cms.api.brightcove.com/v1/accounts/{account_id}/counts/playlists

La réponse a le même format que celle des comptages vidéo.

Vous pouvez également utiliser le paramètre de recherche dans les demandes de nombre de listes de lecture. Ainsi, par exemple, vous pouvez savoir combien de listes de lecture manuelles sont dans le compte en ajoutant q=type:EXPLICIT à la demande.

De plus, vous pouvez obtenir le nombre de vidéos dans n'importe quelle liste de lecture manuelle ou intelligente en faisant votre demande GET à :

    https://cms.api.brightcove.com/v1/accounts/{account_id}/counts/playlists/playlist_id/videos
