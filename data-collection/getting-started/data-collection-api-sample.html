Remarques sur l'échantillon

Le code de cet exemple est entièrement en JavaScript et comprend deux parties (toutes deux trouvées dans le CodePen : le script du plug-in est en haut et un court morceau de code qui ajoute le plug-in au lecteur en bas) :

  • Le script qui ajoute le plugin au lecteur videojs
  • Le script du plugin qui envoie les données au Data Collector

Notez que la façon dont les données sont « envoyées » à l'API de collecte de données consiste à définir l'URL de la demande comme src valeur pour une nouvelle image (img ), puis en injectant cet élément dans la page.

Vous devriez utiliser la même méthode plutôt que XMLHttpRequest ou autre afin d'éviter SCRO problèmes.
