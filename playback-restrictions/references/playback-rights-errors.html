Référence d'erreur

La liste d'état et les codes d'erreur suivants vous aideront à déboguer les problèmes lorsque vous travaillez avec les restrictions de lecture Brightcove:

Statut Code d'erreur Sous-code d'erreur La description
400 BAD_REQUEST DUPLICATE_PARAMETERS Le même nom de paramètre a été fourni plusieurs fois dans la demande
401 ACCESS_DENIED TOKEN_REQUIRED UNE Jeton Web JSON (JWT) est requis par les droits de lecture mais n'a pas été fourni
401 ACCESS_DENIED TOKEN_VALIDATION Les Jeton Web JSON (JWT) fourni n'a pas réussi la validation
403 FORBIDDEN ACCOUNT_ID L'ID de compte dans la clé de stratégie ne correspond pas au compte dans la demande d'API
403 FORBIDDEN CLIENT_GEO La lecture de la vidéo est interdite dans la région géographique actuelle ; le message contiendra des informations supplémentaires sur le problème spécifique
403 FORBIDDEN CLIENT_IP La vidéo est restreinte à l'adresse IP actuelle
403 FORBIDDEN DOMAIN La vidéo ne peut pas être lue sur le domaine actuel
403 FORBIDDEN OFFERS Les Jeton Web JSON (JWT) ne permet pas la lecture basée sur les offres disponibles dans la vidéo
403 FORBIDDEN TAGS Les Jeton Web JSON (JWT) ne permet pas la lecture basée sur les balises disponibles dans la vidéo
403 FORBIDDEN VIDEO_NOT_PLAYABLE Pour une demande de vidéo unique, la vidéo existe, mais sa lecture n'est pas autorisée maintenant
403 FORBIDDEN Denied by stream limiting (create) Le flux limité a atteint le nombre maximum de téléspectateurs
403 FORBIDDEN Denied by stream limiting (renew) Un flux limité est déjà regardé par le nombre maximum de téléspectateurs
403 FORBIDDEN Denied by stream limiting Le flux limité a atteint le nombre maximum de téléspectateurs
403 FORBIDDEN Denied by device limiting Le flux limité a atteint le nombre maximum d'appareils enregistrés distincts
404 NOT_FOUND VIDEO_NOT_FOUND La vidéo demandée n'est pas disponible
404 NOT_FOUND PLAYLIST_NOT_FOUND La liste de lecture demandée n'est pas disponible
405 METHOD_NOT_ALLOWED METHOD_NOT_ALLOWED Seuls GET, HEAD et OPTIONS sont autorisés pour cette API
500 SERVER_ERROR SERVER_ERROR Erreur Interne du Serveur
502 SERVER_ERROR ACCOUNT_RETRIEVE_FAILURE Reçu une mauvaise réponse d'un serveur principal
502 SERVER_ERROR VIDEO_RETRIEVE_FAILURE Vidéo indisponible
502 SERVER_ERROR VIDEO_URLS_RETRIEVE_FAILURE URL de flux non disponibles
502 SERVER_ERROR PLAYLIST_RETRIEVE_FAILURE Liste de lecture vidéo indisponible
502 SERVER_ERROR PLAYBACK_RIGHT_RETRIEVE_FAILURE Impossible de récupérer la lecture directement à partir du CDN
502 SERVER_ERROR PLAYLIST_VIDEOS_RETRIEVE_FAILURE Vidéos de la liste de lecture indisponibles
502 SERVER_ERROR LICENSE_RETRIEVE_FAILURE Échec de l'obtention de la licence du serveur de licences pour cette vidéo
502 SERVER_ERROR OFFERS_RETRIEVE_FAILURE Échec de l'obtention des offres de l'API pour cette vidéo
502 SERVER_ERROR RIGHTS_RETRIEVE_FAILURE Échec de la récupération des droits de lecture pour cette vidéo
503 SERVICE_UNAVAILABLE A renvoyé cette réponse d'un serveur principal
504 SERVER_TIMEOUT Soit un serveur principal ou l'un des serveurs sur lesquels ils s'appuient a expiré
