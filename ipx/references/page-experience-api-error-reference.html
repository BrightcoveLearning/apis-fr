Erreurs de l'API d'expérience sur la page

Référence d'erreur de l'API d'expérience sur la page
Erreur Exemples La description
400 BAD_REQUEST 
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "No template specified"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "Specified template does not exist"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "details": [
    "Path `name` is required."
  ]
}
 Habituellement, un problème avec le corps de la requête, tel que l'omission du template ou un autre champ obligatoire, ou en spécifiant un modèle qui n'existe pas
401 UNAUTHORIZED 
[
  {
    "error_code": "UNAUTHORIZED",
    "message": "Permission denied."
  }
]
 Soit aucun jeton d'accès n'a été envoyé avec la demande, soit les informations d'identification du client utilisées pour créer le jeton d'accès n'avaient pas les autorisations appropriées pour la demande
404 NOT_FOUND 
{
  "status": 404,
  "code": "NOT_FOUND",
  "message": "Experience {experience_id} not found"
}
 L'expérience que vous essayez de mettre à jour n'existe pas. Il a peut-être été supprimé ou l'identifiant d'expérience dans la demande est peut-être incorrect.
