Messages d'erreur généraux

Statut Nom Un message Causer
400 BAD_REQUEST Invalid JSON body on request Il y a une erreur dans JSON pour le corps de la requête ; peut être un problème JSON de base ou un type incorrect pour une valeur de champ
400 BAD_REQUEST "error": "{fieldname} field is required Le corps de la demande ne contient pas le champ obligatoire indiqué
400 AD_CONFIG_NOT_FOUND Ad config not found La configuration d'annonce spécifiée dans une demande SSAI est introuvable
400 AD_CONFIG_INACTIVE Ad config is inactive La configuration des annonces spécifiée dans une demande SSAI est inactive
401 UNAUTHORIZED Permission denied. Jeton d'accès OAuth manquant ou non valide. Si vous avez fourni un jeton d'accès, il se peut qu'il n'ait pas la portée appropriée pour cette demande
403 NOT_AVAILABLE Resource is not available. La ressource que vous demandez est temporairement indisponible - il peut s'agir d'une condition temporaire pendant qu'un certain traitement de la vidéo est en cours, mais si le message persiste, contactez l'assistance.
404 NOT_FOUND Resource does not exist. Vous avez demandé une ressource qui n'existe pas. Vous essayez peut-être de mettre à jour une configuration publicitaire ou d'obtenir une trace publicitaire sur une session qui n'existe plus. Cette erreur sera également renvoyée si vous utilisez une méthode HTTP non autorisée sur le point de terminaison.
500 UNKNOWN an unknown internal error occurred Problème dans le système Brightcove - réessayez plus tard.
503 SERVICE_UNAVAILABLE The API is temporarily unavailable Problème de backend - réessayez plus tard.
504 TIMEOUT something took too long Serveur probablement trop occupé - réessayez plus tard.
