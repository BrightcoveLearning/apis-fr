Aperçu

Ce document détaille les opérations disponibles à partir des différentes API. Vous en avez besoin lorsque vous faites des demandes au service OAuth pour informations d'identification du client.

Remarque: pour certaines API, les informations d'identification du client peuvent également être obtenues à l'aide de l'interface utilisateur des informations d'identification OAuth. Consultez le document suivant pour obtenir des instructions sur l'utilisation de l'interface utilisateur OAuth.

  • Gestion des informations d'authentification API
